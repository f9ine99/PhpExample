# PHP Auth Demo

"this is a simple auth demonistration for php class"

This project is a basic authentication demo built with PHP and MySQL.

## Quick Start

1. Configure your database credentials in `.env`.
2. Run migration:
   `php scripts/migrate.php`
3. Start local server:
   `php -S localhost:8000 -t public`
4. Open:
   `http://localhost:8000`
